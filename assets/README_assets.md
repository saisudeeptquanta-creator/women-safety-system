# Assets

The application references two optional binary assets:

- **siren.mp3** — played automatically when an SOS alert is triggered
  (`pages/5_SOS_Alert.py`). Drop any short alarm/siren MP3 here with this
  exact name. The app works fine without it (the audio is simply skipped).
- **logo.png** — optional logo used for branding. Add any square PNG.

Free sources for a siren sound: https://pixabay.com/sound-effects/search/siren/

`style.css` in this folder is the live application theme and is required.
